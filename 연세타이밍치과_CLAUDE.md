## 1. 프로젝트 개요

### 1.1 배경
- 연세타이밍치과(ystiming.com)의 기존 홈페이지가 2020년 제작 이후 노후화
- 클라이언트가 연세다감치과(yonseidagamdental.com) 수준의 전면 개편 요청
- 기존 v1~v7 프로토타입에서 인트로 모션은 확정, 내부 디자인을 다감치과 스타일로 전환

### 1.2 핵심 방향
- **디자인**: 카페24 한국형 상업 스타일 (AI 느낌 완전 배제)
- **구조**: 다감치과 7페이지 체계를 1:1 매핑
- **인트로**: 기존 확정된 시네마틱 인트로 유지 (검은 배경 + "정확한 진단, 정직한 진료" 타이포 모션)
- **네비게이션**: 메뉴 클릭 시 별도 페이지 이동 (스크롤 다운 아님)
- **산출물**: CLAUDE.md + React/Next.js 코드 + HTML/CSS 카페24 빌더용

### 1.3 기술 스택
- **Primary**: Next.js 14 App Router + TypeScript + Tailwind CSS
- **Secondary (카페24)**: HTML5 + CSS3 + Vanilla JS (Swiper.js, AOS.js)
- **폰트**: Pretendard Variable (본문) + 포인트 세리프 (섹션 타이틀 영문)
- **호스팅**: 카페24 빌더 (최종 배포용)

## 4. 디자인 시스템

### 4.1 절대 금지 사항
- 파티클 효과, AI 느낌 애니메이션
- 그라데이션 배경 (단색 또는 실사진만)
- 카드형 균등 그리드 (비대칭 레이아웃 지향)
- 이모지
- 일러스트/3D 아이콘 (SVG 라인 아이콘만 허용)
- Inter, Roboto, Arial 등 범용 폰트

### 4.2 컬러 시스템
```css
:root {
  /* Primary */
  --color-navy: #0A1628;
  --color-navy-mid: #0F2035;
  --color-navy-light: #162A45;
  
  /* Accent - 클라이언트 확인 후 확정 */
  --color-accent: #C8A96E;        /* 골드 (기존 v7 기준) */
  --color-accent-light: #D4B87A;
  
  /* Neutral */
  --color-white: #FFFFFF;
  --color-off-white: #F5F5F0;     /* 다감치과 오프화이트 배경 */
  --color-gray-100: #E8ECF1;
  --color-gray-400: #8494A7;
  --color-gray-600: #4B5C6F;
  --color-dark: #111827;
  --color-black: #000000;
}
```

### 4.3 타이포그래피
```css
/* 본문 */
font-family: 'Pretendard Variable', 'Pretendard', sans-serif;

/* 영문 섹션 타이틀 */
font-family: 'Cormorant Garamond', serif;  /* 또는 클라이언트 피드백 반영 */

/* 포인트 헤딩 (한글) */
font-weight: 700;
letter-spacing: -0.02em;
```

### 4.4 섹션 배경 교차 패턴 (다감치과 패턴 그대로)
```
흰색(#FFF) → 오프화이트(#F5F5F0) → 다크(#0A1628) → 흰색 → ...
```
- 각 섹션 사이에 배경색을 교차 배치하여 시각적 리듬감 형성
- 다크 섹션에서는 흰색/골드 텍스트 사용
- 사진이 많은 섹션은 오프화이트 배경 위에 배치

### 4.5 인터랙션/모션
- **스크롤 Reveal**: AOS.js (fade-up, fade-in, zoom-in) — 각 섹션 진입 시 요소별 순차 등장
- **슬라이더**: Swiper.js (히어로 배너, B&A, 진료과목, 내부시설)
- **호버**: 부드러운 scale(1.02) + box-shadow 확대
- **페이지 전환**: 없음 (카페24 멀티페이지 기준)
- **인트로 모션**: 별도 구현 (검은 배경 → 타이포 애니메이션 → 메인 페이지 전환)

## 14. 참고 사이트 핵심 코드 패턴

### 14.1 다감치과 헤더 동작
```
스크롤 Y = 0: 투명 배경, 흰색 로고+메뉴
스크롤 Y > 100: 흰색 배경, 어두운 로고+메뉴, box-shadow
메가메뉴: 각 메뉴 호버 시 sub-menu 드롭다운 (하위 앵커 목록)
```

### 14.2 다감치과 히어로 슬라이더
```
Swiper: effect "fade", autoplay 5000ms, pagination "bullets"
각 슬라이드: background-image cover + 텍스트 absolute 오버레이
```

### 14.3 다감치과 의료진 무한 텍스트 루프
```
CSS: @keyframes marquee { 0% { transform: translateX(0); } 100% { transform: translateX(-50%); } }
텍스트 2배 복제하여 무한 스크롤 효과
```
