### 5.2 공통 페이지 구조 패턴

#### 5.2.1 서브페이지 헤더 (sub_head)
모든 서브페이지 공통:
```
[전체 너비 히어로 이미지] - 스케일 애니메이션 (1.13 → 1.0)
  ├─ [중앙 정렬 텍스트 오버레이]
  │    ├─ h3: 아이브라우 (선택적, ex: "의식하진정법")
  │    ├─ h2: 페이지 타이틀 ("연세다감의 수면 치료 센터")
  │    └─ p: 부제목 설명
  └─ [하단 서브 네비게이션 바]
       └─ 가로 정렬 앵커 링크 (hover 시 상단 보더)
       └─ 스크롤 시 fixed 변환 + backdrop-filter blur
```

#### 5.2.2 배경색 교차 패턴
섹션별 배경색이 규칙적으로 교차:
```
흰색(#fff) → 라이트그레이(#f5f5f5) → 흰색 → 다크블루(#617ca5 또는 #171717) → 흰색 → ...
```
- **부분 배경**: `::before` 가상 요소로 섹션 하단 일부만 배경색 적용 (implant01, sleep03)
- **투명도 배경**: `rgba(143, 167, 203, 0.2)` 같은 반투명 블루 배경

#### 5.2.3 섹션 패딩 패턴
- **일반 섹션**: padding-top 130~190px, padding-bottom 110~190px
- **매우 넓은 간격**: 섹션 간 총 300~370px 여백 (하단 pb + 상단 pt)
- **sub_head 포함 시**: padding-top 0 (히어로 이미지가 대신)

### 5.4 핵심 디자인 패턴 요약

#### 5.4.1 이미지+텍스트 배치 패턴 (가장 중요)

사이트 전체에서 **3가지 주요 패턴**이 반복:

**패턴 A: 이미지-좌 + 텍스트-우 (가장 빈번)**
- `sleep02`: 수면마취 설명
- `sleep07`: 추천 대상 (이미지 + 리스트)
- `implant03`: 네비게이션 임플란트 (이미지 + 리스트)
- `implant05`: 전악 임플란트 (이미지 + 텍스트)
- `implant06`: 재수술 (이미지 + 텍스트)
- `implant07`: 재수술 상세 (인터랙티브 이미지 + 리스트)
- `intro01`: 5가지 약속 (이미지 + 리스트)
- `beauty01`: 라미네이트 (Swiper + 리스트)

**패턴 B: 텍스트-좌 + 이미지-우**
- `main_intro`: 치과 소개
- `beauty03`: 앞니 레진 (텍스트+이미지 타이틀 영역)
- `main_sleep`: 주요 진료과목 슬라이드

**패턴 C: 중앙 정렬 (텍스트 위 + 콘텐츠 아래)**
- `implant01`, `implant02`, `implant04`: 중앙 텍스트 → 이미지/인포그래픽
- `sleep03`, `sleep04`, `sleep05`: 중앙 텍스트 → 카드/인포그래픽
- `intro02`: 의료진 중앙 정렬

> **좌우 교차(alternating) 패턴은 사용하지 않음** - 같은 방향(주로 이미지-좌)이 반복됨

#### 5.4.2 "추천 대상" / "장점" 표현 패턴

**카드 그리드를 사용하지 않음.** 대신 2가지 패턴:

**패턴 1: 이미지 + 세로 리스트** (sleep07, implant03, implant07)
```
[큰 이미지]  [번호 리스트]
             01 텍스트텍스트
             02 텍스트텍스트
             03 텍스트텍스트
             04 텍스트텍스트
```
- flex 가로 배치 (이미지 좌, 리스트 우)
- 리스트: flex-direction column, 각 항목 배경색(흰색 or #dedede)
- 항목: rounded 14px, 번호 아이콘 SVG + 텍스트
- hover 인터랙션: 좌측 이미지 변경 + 항목 색상 전환

**패턴 2: 3열 가로 포인트** (implant01, implant04, sleep03, implant06)
```
    [01]          [02]          [03]
  번호이미지    번호이미지    번호이미지
  설명텍스트    설명텍스트    설명텍스트
```
- flex, space-around / space-between
- 세로 정렬: 번호 이미지(PNG) → 텍스트
- AOS fade-up, 순차 delay (0, 100ms, 200ms)

#### 5.4.3 배경색 사용 규칙

| 배경색 | 용도 | 적용 섹션 |
|--------|------|-----------|
| #fff (흰색) | 기본 배경, 대부분의 섹션 | implant01, 02, beauty01~05 |
| #f5f5f5 | 교차 구분, 중립 배경 | implant03, sleep05, main_bna |
| #e8ebf0 | 부분 배경 (::before) | implant01 하단, sleep03 하단 |
| rgba(143,167,203,0.2) | 투명 블루 | implant04 하단 |
| #617ca5 (sub-color) | 강조 블루 배경 | implant05 전체, sleep07 전체 |
| #171717 | 다크 배경 | sleep06 |
| #f9fbff | 아주 연한 블루 | sleep08 내부 래퍼 |

#### 5.4.4 애니메이션 패턴

- **AOS 라이브러리** 사용 (자체 IntersectionObserver 아님)
- **주요 효과**: `fade-up` (대부분), `fade-left` (리스트 항목)
- **duration**: 1000ms 통일
- **delay**: 순차 적용 (0, 100, 200ms)
- **히어로 이미지**: scale(1.13→1.0), 3s animation
- **Swiper 전환**: speed 1000ms, fade/slide 효과

#### 5.4.5 컴포넌트 재사용 패턴

- **번호 아이콘**: `list01.svg ~ list06.svg` (01~06 둥근 배지)
- **번호 박스 이미지**: `sleep03_con01_box.png ~ con03_box.png` (01~03 큰 번호)
- **화살표 버튼**: `white_btn.svg`, `black_btn.svg` (CTA 링크용)
- **main_bna 섹션**: 메인/임플란트 페이지에서 동일 구조 재사용
- **main_info 섹션**: 모든 페이지 하단에 동일 구조 재사용 (지도+진료시간)
- **quick_menu**: 모든 페이지 우측 고정 사이드바
- **main_form**: 모든 페이지 빠른상담 폼

#### 5.4.6 독특한 레이아웃 패턴 (페이지별)

**메인 페이지만의 패턴:**
- 히어로: 100vh + fade Swiper + 진입 애니메이션 체인
- 의료진: marquee 배경 텍스트 + z-index 레이어링
- 진료과목 슬라이더: 텍스트+이미지 2단 fade Swiper

**임플란트 페이지만의 패턴:**
- 인터랙티브 이미지 (hover→src 변경): implant07
- SVG 인포그래픽 전체 너비: implant02

**수면 페이지만의 패턴:**
- 텍스트만 인트로 (sleep01): 회색 텍스트 + 블루 강조
- 수평 스크롤 장비 카드 (sleep06): slidesPerView 2.5
- 캡슐형 주의사항 (sleep08): 보더 라벨 + 인라인 설명

**심미 페이지만의 패턴:**
- 세로 Swiper (beauty01): direction vertical
- 태그 뱃지 배열 (beauty04, beauty05): 가로 나열 아이콘+텍스트
- 세로 카드 스택 (beauty04): 이미지좌+텍스트우, 4개 순차 배열

**소개 페이지만의 패턴:**
- 탭 기반 이미지 전환 (intro05): 4개 탭 → 4개 Swiper
- 비디오 모달 (intro04): Vimeo embed, 모달 팝업

### 5.5 우리 프로젝트에 적용할 핵심 인사이트

1. **이미지-텍스트 배치**: 주로 이미지-좌 + 텍스트-우 방향 유지 (좌우 교차 아님)
2. **추천 대상 섹션**: 카드 그리드 대신 → 대형 이미지 + 세로 번호 리스트 (hover 인터랙션)
3. **장점/특징**: 3열 가로 포인트 (번호 이미지 + 짧은 텍스트)
4. **배경색**: 흰색 기본, #f5f5f5 교차, 강조 섹션은 브랜드색 배경
5. **타이포 계층**: 아이브라우(소) → 대제목(42~52px) → 설명(20px) 3단계
6. **패딩**: 섹션 간 130~190px 상하 패딩 (넉넉한 여백)
7. **AOS 애니메이션**: fade-up + fade-left, duration 1000ms, delay 순차
8. **Swiper 활용**: 히어로(fade), B&A(slide), 장비(slide), 라미네이트(vertical)
9. **공통 하단**: 진료안내(지도+시간) + 퀵메뉴 + 상담폼 전 페이지 동일
